## 0. 引用与标注规范（最小）
- 术语引用：在正文首次出现时写全：`[TERM-XXX] 中文术语`（TERM 来自 `docs/standards/ssot-glossary.md`）。
- PRD 需求引用：当测试覆盖某条 PRD 需求时，句末标注 `（需求：MOD-xxx）`。
- 技术契约引用：当测试覆盖某项技术契约时，引用“文件相对路径 + 章节号”（例如：`docs/technical/.../xxx.md#x.y`）。
- 规则引用：若 PRD 中存在 R# 规则，统一 `（规则：R#）`。
